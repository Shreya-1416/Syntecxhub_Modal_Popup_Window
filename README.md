# React Modal Popup Window

A modern, responsive **Modal / Popup Window** built using **React** and **Tailwind CSS**, featuring smooth animations, dark mode, and mobile-friendly design.

This project demonstrates clean component architecture, state management, and UI polish suitable for real-world applications, portfolios, and internship submissions.

---

## Features

- Reusable React Modal component
- Smooth open & close animations
- Dark / Light mode toggle (persistent with localStorage)
- Fully responsive & mobile-friendly
- Click outside & ESC key to close
- Background scroll lock when modal is open
- Styled using Tailwind CSS (utility-first)

---

## Tech Stack

- **React**
- **Tailwind CSS**
- **JavaScript (ES6+)**
- **HTML5**

## Getting Started

### 1️ Clone the repository
```bash
git clone https://github.com/your-username/react-modal.git
cd react-modal
```
## 2️ Install dependencies
```bash
npm install
```

## 3️ Run the project
```bash
npm start
```

## Dark/Light Mode

- Toggle using the button in the top-right corner
- Preference is saved using localStorage
- Automatically restored on page refresh

## Animations & UX

- Fade + scale modal animations
- Backdrop blur for better focus
- Optimized for both desktop and mobile devices
