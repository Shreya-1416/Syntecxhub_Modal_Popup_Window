# React Modal Popup Window

A modern, responsive **Modal / Popup Window** built using **React** and **Tailwind CSS**, featuring smooth animations, dark mode, and mobile-friendly design.

This project demonstrates clean component architecture, state management, and UI polish suitable for real-world applications, portfolios, and internship submissions.
